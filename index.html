TextSuggest

Save keystrokes. Use autocomplete, expansions, math results, commands and more in any application.

A utility to autocomplete words in the GUI.

Licensed under the GNU GPL 3. TextSuggest is free as in freedom.

Install

Ubuntu and Debian

Get the deb file: Download textsuggest-git.deb

Arch Linux

Get it from the AUR: textsuggest-git

Manual

Run sudo ./install.sh.

Using TextSuggest

Simply bind the command textsuggest to a keyboard shortcut. Type a word, select it, press the shortcut. See TextSuggest give you suggestions. You don't even have to select the word: for that, bind the shortcut to textsuggest --auto-selection. You can also use textsuggest --all-words for getting all words known.

Advanced usage

TextSuggest is flexible and has many options. See the README for full documentation.
